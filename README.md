# 📦 Supply Chain Performance Dashboard

🚀 **End-to-End Data Analyst Project | Python + SQL + Power BI**

---

## 📊 Project Overview

This project presents a **comprehensive Supply Chain Analysis Dashboard** designed to monitor and optimize business performance across revenue, suppliers, products, and logistics.

The dashboard enables stakeholders to gain **actionable insights** into operational efficiency, profitability, and supply chain bottlenecks.

---

## 🎯 Business Problem

Organizations often struggle with:

* Inefficient supplier performance
* Delays in shipping and logistics
* Lack of visibility into product-level performance
* Increasing defect rates affecting quality

👉 This project solves these problems by building a **centralized analytical dashboard**.

---

## 🛠️ Tech Stack

* **Python** → Data Cleaning & Preprocessing (Pandas, NumPy)
* **SQL (PostgreSQL)** → Data Transformation & Querying
* **Power BI** → Interactive Dashboard & Visualization
* **Excel/CSV** → Raw Dataset
